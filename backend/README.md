# 🎓 AlumniConnect++

A full-stack alumni networking platform built using React, Node.js, Express, and MongoDB.

## 🚀 Project Overview

AlumniConnect++ allows students and alumni to authenticate using LinkedIn OAuth and view registered alumni profiles.

The platform stores verified user data securely in MongoDB and displays alumni information in a clean frontend interface.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- LinkedIn OAuth (OpenID Connect)

---

## 🔐 Features Implemented

- LinkedIn Login (OAuth 2.0)
- MongoDB Database Integration
- Alumni Data Storage
- REST API (`/api/alumni`)
- Frontend Data Fetching
- Alumni List Display
- Role-ready structure for Student / Alumni separation
- Environment Variable Configuration using dotenv
